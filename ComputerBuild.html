Computer Build

This project entailed buying computer components from Canada Computers and building a PC. So what we did, is we looked on Canada Computers website to find parts that were compatible and fit in our budget. We had a small compatability issue, but once we sorted it out we were able to complete the builds. I had a lot of fun doing this build and I learned the importance of a budget, how to build a computer, and how to buy parts that are compatible with eachother.
